Warm up

We've been cloning a lot of repos for our warm ups. Today I want to make sure you know how to create your own files while working with JavaScript

Warm up steps

  1. Make a new folder on your computer for this warm up
  2. Create an index.html file with your typical document structure
  3. Add an h1 element with "Hello, world" in your index file
  4. Create a js folder
  5. Create a script.js file in the js folder
  6. Add a jQuery CDN script tag to your index file
  7. Connect your javascript to your index file
  8. In the js file, make a variable called university with the value "The New School"
  9. In the js file, make a variable called currentYear with the current year as a number
  10. Make a variable called yearsUntilGraduation with the number of years until graduation
  11. Make a variable called graduationYear using the two steps above to make that variable
  12. Make a variable called message that has the format of "I will graduate from The New School in 2018" using the variables you created
  13. Swap out the text in your h1 element using jQuery
  14. Change some of your variables to see how the console log message changes
